# Halo House Prototype

React + Tailwind CSS prototype for a beauty-services marketplace focused on safer discovery, provider verification, and streamlined booking.

## Included screens

- Discovery hero and high-level user flow
- Sample provider listing
- Stylist and client profile cards
- Booking and payment placeholder
- Verification status page
- Tailwind CSS styling with Vite/PostCSS

## Run locally

```bash
npm install
npm run dev
```

## Notes

- This is a front-end prototype for pitching and product direction.
- Booking payments and verification are visual placeholders only.
